# Mobile Gallery

기기 사진 라이브러리를 읽어 내 사진을 보여주는 Expo React Native 갤러리입니다.
웹(PC)에서는 파일 선택을 통해 사진을 불러옵니다.

## 특징
- 기기 사진 라이브러리 권한 요청
- 2열 그리드
- 파일명 검색
- 사진 상세 모달
- 웹(PC) 파일 선택 지원

## 실행
```bash
npm install
npm run start
```

## 사용 방법
- 모바일: "권한 요청" 버튼을 눌러 사진 접근 허용 후 자동 표시
- 웹(PC): "사진 가져오기" 버튼으로 로컬 파일 선택

## 메모
- 아이콘/스플래시는 임시 1x1 PNG입니다.
